  ```
  <div className="px-3 py-1.5">

      <span>{label}</span>

    </div>
    <input type="range" ... className="w-full" style={{ accentColor: '#3b82f6' }} />
  </div>
  ```

**Slider commit-on-release pattern (CRITICAL -- prevents ghost undo entries):**
- Each slider has a local `useState<number | null>(null)` for in-progress drag display
- `displayValue = localValue ?? storeValue ?? defaultValue`
- `onChange`: sets local state only (NO store write)
- `onPointerUp` + `onTouchEnd`: reads `(e.target as HTMLInputElement).value`, parses it, calls `updateClipSettings(clipId, { fieldName: parsedValue })`, then sets local state to null
- When store value changes (e.g., undo), `localValue` is null so display follows store

**Slider parameters:**
| Slider | min | max | step | default | parse |
|--------|-----|-----|------|---------|-------|
| Blur | 0 | 10 | 1 | 0 | parseInt(v, 10) |
| Brightness | -1.0 | 1.0 | 0.05 | 0 | parseFloat(v) |
| Contrast | 0.0 | 2.0 | 0.05 | 1.0 | parseFloat(v) |
| Saturation | 0.0 | 3.0 | 0.05 | 1.0 | parseFloat(v) |

**Crop inputs section:**
- Section label: same styling as Filters but text "Crop"
- Four number inputs in a 2x2 grid: X, Y, Width, Height
- Each input: `<input type="number" min={0} className="w-full bg-zinc-800 border border-zinc-700 rounded px-2 py-1 text-xs text-white">`
- Label above each input: `<label className="text-xs text-zinc-400">`
- Default display values when `settings?.crop` is null: X=0, Y=0, Width=`clip.sourceWidth`, Height=`clip.sourceHeight`
- On change: parse with `parseInt(value, 10)`, guard NaN, then call `updateClipSettings(clipId, { crop: { x, y, width, height } })` using current values for unchanged fields
- When crop is null and user first edits, create the full crop object using current display defaults for unchanged fields

**Resize inputs section:**
- Section label: same styling but text "Resize"
- Two number inputs side by side: Width, Height
- Aspect ratio lock toggle button using lucide-react `Lock` / `Unlock` icons:
  ```typescript
  import { Lock, Unlock } from 'lucide-react'
  ```
  - Local `useState<boolean>(true)` for lock state (ON by default per CONTEXT.md)
  - Button: `<button onClick={() => setAspectLocked(!aspectLocked)} className="p-1 rounded hover:bg-zinc-700">`
  - Show `<Lock size={14} />` when locked, `<Unlock size={14} />` when unlocked
- Default display when `settings?.resize` is null: Width=`clip.sourceWidth`, Height=`clip.sourceHeight`
- When locked and width changes: `newHeight = Math.round(newWidth / (clip.sourceWidth / clip.sourceHeight))`
- When locked and height changes: `newWidth = Math.round(newHeight * (clip.sourceWidth / clip.sourceHeight))`
- Source aspect ratio always computed from `clip.sourceWidth / clip.sourceHeight` (NOT from current resize values)
- Guard division by zero: if sourceHeight is 0, treat as unlocked
- On change: `updateClipSettings(clipId, { resize: { width: newWidth, height: newHeight } })`
