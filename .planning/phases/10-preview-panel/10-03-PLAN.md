    Create `src/hooks/usePreview.ts` — a hook that:

    **Accepts:** `canvasRef: React.RefObject<HTMLCanvasElement | null>` parameter.

    **Manages video elements** (one per video-track clip):
    - `videoElementsRef = useRef<Map<string, HTMLVideoElement>>(new Map())`
    - Subscribe to store clip changes. When a new video clip appears: `document.createElement('video')`, set `src = URL.createObjectURL(clip.sourceFile)`, `preload = 'auto'`, `muted = false` (video audio plays naturally per user decision), hide it, append to document.body. Store in the Map keyed by clip.id.
    - When a clip is removed: `URL.revokeObjectURL(el.src)`, `el.remove()`, delete from Map.
    - On hook unmount: clean up all video elements and revoke all object URLs.

    **Manages audio elements** (one per audio-track clip):
    - `audioElementsRef = useRef<Map<string, HTMLAudioElement>>(new Map())`
    - Same lifecycle as video elements but using `new Audio()` (or `document.createElement('audio')`).
    - Audio elements are for clips on the `audio` track ONLY. Video clips play audio through their video elements.
    - Set `el.volume = Math.min(1, settings.volume)` (HTMLAudioElement.volume is 0-1; values > 1.0 are clamped — known preview limitation).

    **rAF loop** (start/stop based on `isPlaying`):
    - `rafRef = useRef<number>(0)`
    - `lastRealTimeRef = useRef<number>(0)`
    - Subscribe to `isPlaying` via `useStore((s) => s.ui.isPlaying)` (React subscription to trigger effect).
    - `useEffect` watching `isPlaying`:
      - If `isPlaying === true`: set `lastRealTimeRef.current = 0`, call `requestAnimationFrame(tick)`.
      - If `isPlaying === false`: `cancelAnimationFrame(rafRef.current)`, call `drawFrame(useStore.getState().ui.playheadTime)` to show current frame on pause.
      - Return cleanup: `() => cancelAnimationFrame(rafRef.current)`.

    **tick function:**
    ```typescript
    function tick(realTime: DOMHighResTimeStamp) {
      const store = useStore.getState()
      if (!store.ui.isPlaying) return

      if (lastRealTimeRef.current === 0) {
        lastRealTimeRef.current = realTime
        rafRef.current = requestAnimationFrame(tick)
        return
      }

      const realDt = (realTime - lastRealTimeRef.current) / 1000
      const dt = Math.min(realDt, 1 / 15)  // clamp to prevent tab-blur jumps
      lastRealTimeRef.current = realTime

      const totalDuration = computeTotalDuration(store.clips)
      const newTime = store.ui.playheadTime + dt

      if (newTime >= totalDuration) {
        store.setPlayheadTime(totalDuration)
        store.setIsPlaying(false)
        drawFrame(totalDuration)
        return
      }

      store.setPlayheadTime(newTime)
      drawFrame(newTime)
      rafRef.current = requestAnimationFrame(tick)
    }
    ```

    **drawFrame function:**
    - Get canvas context from `canvasRef.current`.
    - Size the canvas: `canvas.width = canvas.offsetWidth * devicePixelRatio`, `canvas.height = canvas.offsetHeight * devicePixelRatio`. This ensures crisp rendering on retina.
    - Clear canvas with black: `ctx.fillStyle = '#000'`, `ctx.fillRect(0, 0, canvas.width, canvas.height)`.
    - Find active video clip: `findClipAt(store.clips, store.tracks, 'video', time)`.
    - If no active video clip, return (black canvas = gap).
    - Get the video element from `videoElementsRef`.
    - Get clip settings: `store.clipSettings[clip.id]` or use defaults if absent.
    - Compute clip-local time: `clip.trimStart + (time - clip.startTime)`.
    - **During playback** (`isPlaying=true`): Let the video element play naturally. If the video is paused, call `el.play().catch(() => {})`. Only seek if drift > 0.3s: `if (Math.abs(el.currentTime - clipLocalTime) > 0.3) el.currentTime = clipLocalTime`.
    - **During scrub** (`isPlaying=false`): Set `el.currentTime = clipLocalTime` and listen for `seeked` event before drawing. Use `el.addEventListener('seeked', drawFn, { once: true })` pattern. If already at the correct time (drift < 0.01), draw immediately.
    - Apply canvas transforms and draw:
      ```typescript
      const cw = canvas.width
      const ch = canvas.height
      const settings = store.clipSettings[clip.id] ?? defaultSettings
      const rot = settings.rotation
      const effectiveSrcW = (rot === 90 || rot === 270) ? clip.sourceHeight : clip.sourceWidth
      const effectiveSrcH = (rot === 90 || rot === 270) ? clip.sourceWidth : clip.sourceHeight
      const scale = Math.min(cw / effectiveSrcW, ch / effectiveSrcH)
      const dw = effectiveSrcW * scale
      const dh = effectiveSrcH * scale

      // Source rect (crop)
      const sx = settings.crop?.x ?? 0
      const sy = settings.crop?.y ?? 0
      const sw = settings.crop?.width ?? clip.sourceWidth
      const sh = settings.crop?.height ?? clip.sourceHeight

      ctx.filter = buildCanvasFilter(settings)
      ctx.save()
      ctx.translate(cw / 2, ch / 2)
      if (rot !== 0) ctx.rotate((rot * Math.PI) / 180)
      if (settings.flipH) ctx.scale(-1, 1)
      if (settings.flipV) ctx.scale(1, -1)
      ctx.drawImage(el, sx, sy, sw, sh, -dw / 2, -dh / 2, dw, dh)
      ctx.restore()
      ctx.filter = 'none'
      ```

    A thin rendering component that reads store state and renders canvas + controls bar.

    ```tsx
    import { useRef } from 'react'
    import { useStore } from '../store'
    import { usePreview } from '../hooks/usePreview'
    import { formatTimecode, computeTotalDuration } from '../utils/previewUtils'

    export function PreviewPanel() {
      const canvasRef = useRef<HTMLCanvasElement>(null)
      usePreview(canvasRef)

      const playheadTime = useStore((s) => s.ui.playheadTime)
      const isPlaying = useStore((s) => s.ui.isPlaying)
      const clips = useStore((s) => s.clips)
      const setIsPlaying = useStore((s) => s.setIsPlaying)
      const setPlayheadTime = useStore((s) => s.setPlayheadTime)

      const hasClips = Object.keys(clips).length > 0
      const totalDuration = computeTotalDuration(clips)

      const handleTogglePlay = () => {
        if (!hasClips) return
        // If at end of timeline and pressing play, restart from beginning
        if (!isPlaying && playheadTime >= totalDuration && totalDuration > 0) {
          setPlayheadTime(0)
        }
        setIsPlaying(!isPlaying)
      }

      return (
        <div className="flex flex-col h-full bg-zinc-900">
          <div className="flex-1 relative min-h-0">
            <canvas
              ref={canvasRef}
              className="absolute inset-0 w-full h-full"
            />
            {!hasClips && (
              <div className="absolute inset-0 flex items-center justify-center text-zinc-500 text-sm pointer-events-none">
                Import clips to preview
              </div>
            )}
          </div>
          <div className="flex-none flex items-center justify-center gap-4 px-4 py-2 border-t border-zinc-800">
            <span className="text-zinc-400 font-mono text-sm tabular-nums">
              {formatTimecode(playheadTime)}
            </span>
            <button
              onClick={handleTogglePlay}
              className="w-8 h-8 flex items-center justify-center rounded-full bg-zinc-700 hover:bg-zinc-600 text-white transition-colors"
              title={isPlaying ? 'Pause' : 'Play'}
            >
              {isPlaying ? '\u23F8' : '\u25B6'}
            </button>
            <span className="text-zinc-500 font-mono text-sm tabular-nums">
              {formatTimecode(totalDuration)}
            </span>
          </div>
        </div>
      )
    }
    ```

    Key details:
    - Controls bar layout matches user decision: `[MM:SS] [play/pause] [MM:SS / total]`
    - Timecode is MM:SS per user decision (formatTimecode handles this)
    - Empty state: dark canvas + "Import clips to preview" hint text (not EmptyState component — per user decision)
    - Play button shows unicode pause (U+23F8) when playing, play (U+25B6) when paused
    - When at end of timeline and play pressed, reset playhead to 0 before playing (user expects restart)
    - `tabular-nums` class for monospace number alignment in timecode
    - `min-h-0` on canvas container to prevent flex overflow
